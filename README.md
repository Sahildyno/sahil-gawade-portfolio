## 🧪 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/Sahildyno/sahil-gawade-portfolio.git
cd sahil-gawade-portfolio
2. Install dependencies

npm install
# or
yarn install
3. Run the development server

npm run dev
# or
yarn dev
The app will run at http://localhost:5173.
